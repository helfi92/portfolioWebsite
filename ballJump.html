BALL JUMP APPLET

If the Java applet is blocked by your security settings, this is because Java has further enhanced security to make the user system less vulnerable to external exploits. Scroll down to the end of the page for instructions on how to fix this issue.

Bounce the ball left and right on the platforms. Occasionally a flying coloured ball will appear and will give you some special force. Make sure you don't go on a red ball because it will increase the gravity of the ball, hence it will be harder to bounce high. The longer you stay alive, the higher score you will have.

Instructions to make the game work

  1. Go to the Java Control Panel (On Windows Click Start and then Configure Java)
  2. Click on the Security tab
  3. Click on the Edit Site List button
  4. Click Add in the Exception Site List window
  5. Click in the empty field under the Location field and enter this URL: http://hassan.akeel.me/ballJump.html
  6. Re-run this current link and you are now in a position to play this awesome game!
